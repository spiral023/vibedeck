> **Hinweis**: Dieser Artikel beschreibt den Kurs [Claude Code for Everyone](https://ccforeveryone.com/) von Carl Vellotti.

Dies ist der einzige Kurs, der **innerhalb** von Claude Code unterrichtet wird. Er wurde für Menschen ohne technischen Hintergrund entwickelt, die KI effektiv für ihre tägliche Arbeit nutzen wollen.

## Das Konzept: Learning by Doing

Anstatt Videos anzuschauen oder lange Dokumentationen zu lesen, lernst Du hier durch echte Interaktion.
1.  **Installieren**: Claude Code installieren (ca. 15 Min).
2.  **Download**: Kursmaterialien herunterladen.
3.  **Starten**: Den Befehl `/start-1-1` im Terminal eingeben.
4.  **Interaktion**: Claude führt Dich interaktiv durch jede Lektion mit echten Dateien.

## Kursinhalte

### Modul 1: Fundamentals (~3 Stunden)
*   **Dateioperationen**: Dokumente lesen, transformieren und mit dem `@`-Symbol erstellen.
*   **Visual Workspace**: Split-Screen Workflow einrichten, um Claude bei der Arbeit zuzusehen.
*   **Parallel Agents**: Mehrere Claude-Instanzen gleichzeitig nutzen.
*   **Custom Sub-Agents**: Permanente KI-Berater mit unterschiedlichen Perspektiven erstellen.
*   **Project Memory**: Nutzung der `CLAUDE.md`, damit Claude den Kontext behält.
*   **Befehle & Shortcuts**: Slash-Commands und Power-User-Features.

### Modul 2: Vibe Coding (~1-2 Stunden)
*   **Setup & Mindset**: Der Vibe-Coding-Ansatz – Du beschreibst, Claude baut.
*   **Planning**: Anforderungen in einem "Interview-Prozess" definieren, bevor gebaut wird.
*   **Build & Iterate**: Apps scaffolden und mit Screenshots verfeinern.
*   **Version Control**: Arbeit via GitHub sichern.
*   **Go Live**: Deployment auf Vercel mit einer echten URL.

## Voraussetzungen
*   **Abo**: Claude Pro oder Max Abonnement ($20/Monat).
*   **Hardware**: Mac, Windows oder Linux Computer.
*   **Zeit**: Ca. 3 Stunden für das erste Modul.
*   **Vorkenntnisse**: Keine Coding- oder Terminal-Erfahrung erforderlich.
